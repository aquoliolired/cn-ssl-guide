# cn-ssl-guide

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，而是作为一系列与 SSL/TLS 相关参考内容的静态存放与展示空间。

所有页面均为独立的 HTML 文件，可直接在浏览器中打开查看，无需额外后端支持。

## 目录说明

```
cn-ssl-guide/
├── index.html          # 仓库主入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS、图片等）
│   └── style.css
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面存放于此，文件名应具有描述性，便于识别内容主题。
- `assets/`：存放页面共用的样式、脚本或图片等静态资源。
- 仓库根目录下的 `index.html` 可作为导航或介绍页（可选）。

## 页面归档说明

- 每个 HTML 页面均为自包含的完整文档，不依赖外部动态服务。
- 页面内容聚焦于 SSL/TLS 相关的技术说明、配置示例、常见问题等，不包含具体域名或营销信息。
- 页面之间没有强制关联，可独立浏览和引用。
- 归档文件命名应简洁明了，建议使用英文小写字母、数字和连字符。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面或改进现有内容。
- 新增页面请放入 `pages/` 目录，并确保其内容符合仓库主题。
- 请勿在页面中添加任何形式的推广、诱导点击或具体商业引用。
- 如有资源文件更新，请同步修改相关页面中的引用路径。
- 对于过时或错误的内容，建议直接更新或替换对应文件，并记录变更。

本仓库内容仅供学习与参考，不构成任何形式的担保或承诺。具体使用请遵守相关法律法规及开源许可协议（如有）。
